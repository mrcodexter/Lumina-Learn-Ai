🚀 Lumina Learn AI

AI-powered study assistant that helps users generate notes, quizzes, and answers instantly using Google Gemini AI.

---

🌍 Features

- 🧠 AI Chat (Instant Answers)
- 📝 Smart Notes Generator
- ❓ Quiz Generator (MCQ)
- 🌐 Bangla + English Support
- ☁️ Firebase Integration
